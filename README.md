# Portal - Agro Consciente

💻 Ideia do Projeto: Portal "AgroConsciente"

O objetivo do site pode ser usado tanto para educar a comunidade escolar, tanto os agricultores sobre como o uso de tecnologias (como drones e sensores) e de manejos biológicos pode reduzir a necessidade de defensivos químicos, protegendo a saúde e o solo sem perder a produtividade.

"Eu desenvolvi o Portal Agro Consciente, uma página web criada em HTML e CSS, focada no tema do Agrinho deste ano de 2026. O objetivo do projeto é mostrar como os produtores rurais podem usar a tecnologia (como drones e sensores) e soluções biológicas para reduzir o uso de agrotóxicos. Com isso, o site prova na prática que é possível ter um agro forte, produtivo e que, ao mesmo tempo, protege o meio ambiente e o solo da nossa região."

Os 3 pontos centrais do meu Projeto Agrinho é;

. O que é: Um site informativo e interativo focado em sustentabilidade no campo.
. A tecnologia: Feito do zero usando programação HTML (linguagem de marcação para web).
. O impacto: Mostra como a tecnologia ajuda a produzir mais gastando menos recursos químicos, equilibrando produção e preservação.
